This section will collect common errors and fixes for AtMyApp projects.

## Common checks

- Confirm you ran `atmyapp use` in the current workspace.
- Confirm your schema file is named `atmyapp.schema.ts`, `atmyapp.schema.mts`, `atmyapp.schema.js`, `atmyapp.schema.mjs`, or `atmyapp.schema.json`.
- Use `atmyapp migrate --dry-run --verbose` to inspect generated output without uploading it.
- Make sure the project URL points at the expected AtMyApp project.

If you are stuck, open a discussion in the [AtMyApp repository](https://github.com/At-My-App/atmyapp/discussions).
