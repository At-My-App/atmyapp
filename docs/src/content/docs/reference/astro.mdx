`@atmyapp/astro` provides AtMyApp helpers for Astro projects.

## Install

```bash
pnpm add @atmyapp/astro
```

Astro must already be installed in the project.

## Package exports

- `@atmyapp/astro`
- `@atmyapp/astro/client`
- `@atmyapp/astro/components`

## Peer dependency

The package supports Astro `>=4.0.0`.

## Typical use

Use this package when an Astro project needs AtMyApp-managed metadata, client helpers, or integration-specific behavior.
